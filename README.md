# Accessible Video Player

An accessible video player implementation.

## Project structure

- **src/** – Source code (CSS, JS, assets)
- **tests/** – Unit, integration, and e2e tests
- **docs/** – Documentation, screenshots, test reports, research

## Test video (sample.mp4)

The player expects a test video at `src/assets/videos/sample.mp4`. The file is not in the repo (see `.gitignore`).

**Option A (empfohlen):** Eigenes kurzes Test-Video (30–60 s) erstellen (Smartphone oder Screenrecording), als MP4 (H.264) exportieren.

**Option B:** Blender Demo-Videos (Creative Commons):

- Download: <https://download.blender.org/demo/movies/>
- Empfohlen: „Spring“ (klein, lizenzfrei)

Speichere die gewählte Datei als `src/assets/videos/sample.mp4`. Für Performance-Tests sollte die Datei unter 20 MB bleiben.
