# IMP-43: Axe „Needs Review“ – Manuelle Prüfung

Alle „Needs Review“-Items aus dem Axe-Scan müssen manuell geprüft und dokumentiert werden.

## Vorlage

| Datum      | Regel-ID            | Beschreibung    | Ergebnis | Begründung                    |
| ---------- | ------------------- | --------------- | -------- | ----------------------------- |
| YYYY-MM-DD | z.B. color-contrast | Kontrast prüfen | Pass     | Text hat ausreichend Kontrast |
| …          | …                   | …               | Fail/N/A | …                             |

## Ergebnis-Optionen

- **Pass:** Kriterium erfüllt
- **Fail:** Kriterium nicht erfüllt, muss behoben werden
- **N/A:** Nicht anwendbar (z.B. False Positive, Kontext-abhängig)
