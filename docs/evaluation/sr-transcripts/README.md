# Screenreader-Transkripte

Manuell erstellte Transkripte für die Evaluations-Dokumentation.

## Vorlage

Datei: `sr-transcript-YYYY-MM-DD-[szenario].md`

```markdown
# SR-Transkript: [Szenario]

Datum: YYYY-MM-DD
Screenreader: NVDA/JAWS/VoiceOver
Browser: Chrome/Firefox/Safari

## Schritte und Ansagen

| Aktion      | Erwartete Ansage        | Tatsächliche Ansage | Pass/Fail |
| ----------- | ----------------------- | ------------------- | --------- |
| Tab zu Play | „Abspielen, Button"     | …                   |           |
| Enter       | „Video wird abgespielt" | …                   |           |
```

## Szenarien (aus screenreader-walkthrough.md)

- Video mit Untertiteln starten
- Lautstärke ändern
- Settings öffnen/schließen
- Timeline per Tastatur
